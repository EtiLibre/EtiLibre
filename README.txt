ESTRUCTURA DEL PROYECTO
───────────────────────
EtiLibre/
├── EtiLibre.html          ← Abrí este archivo para usar la app
├── libs/
│   ├── pdf.min.js         ← Librería PDF.js (leer PDFs)
│   ├── pdf.worker.min.js  ← Worker de PDF.js
│   └── pdf-lib.min.js     ← Librería pdf-lib (crear PDFs)
├── instalar-libs.sh       ← Re-descarga libs (Linux/Mac)
├── instalar-libs.bat      ← Re-descarga libs (Windows)
└── README.txt             ← Este archivo

CÓMO USAR (modo offline)
────────────────────────
1. Extraé el ZIP en cualquier carpeta de tu computadora
2. Hacé doble clic en EtiLibre.html
3. Se abre en el navegador — listo, funciona sin internet
